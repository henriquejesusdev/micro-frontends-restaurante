# Restaurante Micro Frontends

Projeto criado para praticar Micro Frontends com React, JavaScript e Webpack Module Federation.

## Aplicações

O projeto possui três aplicações separadas:

- Container App: aplicação principal
- Micro Cardápio: lista pratos disponíveis
- Micro Pedido: exibe os itens adicionados ao pedido

## Tecnologias

- React
- JavaScript
- Webpack 5
- Module Federation
- CSS

## Como rodar

### 1. Rodar o Micro Cardápio

```bash
cd micro-cardapio
npm install
npm start
```
